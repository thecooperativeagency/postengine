TROUBLESHOOTING QUICK START
═══════════════════════════════════════════════════════════════════════════════

"Dashboard shows no data"
  → Check backend running: curl http://localhost:3001/health
  → Should return: {"status":"ok","timestamp":"..."}

"Frontend can't reach backend"
  → Check proxy in vite.config.js
  → Make sure backend port 3001 is correct

"No posts showing up"
  → Check X API key is correct
  → Run: node backend/src/tasks/ingestors.js
  → Check backend logs for errors

"WebSocket not connecting"
  → Check backend URL in frontend .env
  → Open dev tools, check console errors

"Database errors"
  → Check DATABASE_URL format
  → Verify PostgreSQL is running
  → Try: psql $DATABASE_URL
