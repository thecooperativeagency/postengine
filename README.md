# The Cooperative Agency - GA4 Dashboards

Interactive GA4 analytics dashboards for dealership performance tracking.

- **[BMW of Jackson](https://thecooperativeagency.github.io/ga4-dashboards/bmw-jackson-dashboard.html)** — Q1 2026 Performance
- **[Brian Harris BMW](https://thecooperativeagency.github.io/ga4-dashboards/brian-harris-bmw-dashboard.html)** — Q1 2026 Performance

## Metrics Tracked

### Website Traffic & Engagement
1. Sessions/Month
2. Unique Visitors
3. Pages Per Session
4. Bounce Rate
5. Avg Session Duration

### Inventory & Search Performance
10. Inventory Page Views
11. Vehicle Detail Engagement

## Data Source

All data pulled from Google Analytics 4 (GA4) via API.
- Reporting Period: March 2 - April 1, 2026
- Data refreshed daily

## Features

- **Real-time charts** showing daily session trends
- **Performance metrics** with target benchmarks
- **Actionable insights** for each dealership
- **Responsive design** for mobile & desktop
- **Color-coded status indicators** (Good/Warning/At Target)

## How to Update

Data updates can be added by:
1. Pulling fresh GA4 reports
2. Updating metric values in each HTML file
3. Pushing changes to GitHub (auto-deploys to live site)

## Development

Built with:
- HTML5 / CSS3
- JavaScript (vanilla)
- Chart.js for visualizations
